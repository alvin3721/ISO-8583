# ISO-8583

A C++ implementation of the ISO-8583 financial norm (it's ISO).

Supported versions:
	ISO-8583-1:1987
	ISO-8583-1:1993
	ISO-8583-1:2003

==== WARNING ====
This is work in progress. So there are missing functionalities.
See code/README for building and running.

Contributors are welcome.
